Using PhysX Vehicles in an Unreal Engine 4 Project (4.24)

April 20, 2019 by oyagci

Open Unreal Engine and create a new C++ project. Then open your project in visual studio and open the Build.cs file.

Add this to the dependencies list: 


"PhysicsCore", "PhysX", "PhysXVehicles", "PhysXVehicleLib"

Then compile your project.
